# CV / Resume Folder

Drop your CV file here as `resume.pdf`.

**Steps:**
1. Export your CV as a PDF
2. Rename the file to `resume.pdf`
3. Place it in this folder (`assets/cv/resume.pdf`)
4. Commit and push — the site will automatically show the inline preview and enable the download button

The portfolio will detect the file and:
- Show an inline PDF preview in the CV section
- Enable the "Download CV" and "Open in New Tab" buttons throughout the site
